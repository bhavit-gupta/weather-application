## ğŸ“¦ Setup & Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/bhavit-gupta/weather-application.git
   cd weather-application
   ```

2. **Configure API Key:**
   - Get your free API key from [WeatherAPI.com](https://www.weatherapi.com/).
   - Create a file named `src/config.js` (you can copy `src/config.example.js`).
   - Add your key to the file:
     ```javascript
     const API_KEY = "your_api_key_here";
     ```

3. **Run the app:**
   Simply open `src/index.html` in your browser!

4. **(Optional) Rebuild CSS:**
   ```bash
   npm install
   npm run build:css
   ```
